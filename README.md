-------------------------README_FILE------------------------
This Miniature Operating System has been made by Mithil Sadhasivam, a (currently) 12 year old programmer.
In order to get the most out of this operating system, follow the instructions below.
This operating system's main use is to be paired with a simple Raspberry PI/Arduino Uno Q/ some equivalent variant that has capacities to run Python Programs
It is highly suggested to pair this with a TFT Screen, however it can be modified to not need screens.

--------FUNCTIONS---------
This operating system comes with a range of functions, including:

[PROGMAN]: The system's default file manager, can delete files and show your available files
           NOTE: The file will be saved on the computer as real readable files. please store the operating system in a separate folder to 
           ensure that unwanted files are not displayed.         
[CALC]:The system's in-built calculator, with support for simple operations.
[DIR]:Lists the files in that folder.
[EXIT]:Ends all functions of the operating system.
[NOTEPAD]:A simple text editor, capable of creating files on the system, editing and reading them.
[RUN]:A function that allows you to run other Python scripts on the OS directly. It will prompt you for the file name.
        [NOTE]:When prompted for the file name, be sure to add the extension(e.g: .py)
[HELP]: Lists all the commands in the program.


The creator hopes you might find this project as inspiration to build better apps for this OS.
